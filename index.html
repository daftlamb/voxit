Vox It ✦

A voxel composition tool based on Heerich.js. Place/remove cubes on the grid to create isometric vector illustrations.

Add mode — Click to place a voxel.
Subtract mode — Click to remove a voxel.
Click an existing voxel — Stack on top.
Shift + drag — Rotate the view.

Render engine: heerich.js by meodai
